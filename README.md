# Control de Asistencia Escolar

Sistema de gestión de asistencia escolar con soporte para lector de huellas digitales.

## Características

- Registro de estudiantes y tutores
- Marcado de asistencia (manual y por huella digital)
- Reportes semanales en PDF
- Panel de control con estadísticas
- Gestión de usuarios

## Tecnologías

- Node.js + Express
- MySQL (Aiven.io)
- EJS templates
- Bootstrap 5
- PDFKit

## Configuración local

1. Clonar el repositorio
2. `npm install`
3. Configurar `.env` con tus credenciales
4. `npm run dev`

## Despliegue

Desplegado en Render.com con base de datos en Aiven.io
